CSS Classes

  • Container Classes

    1. .row - flex row container layout. Distributes it's child elements in a row layout.
    2. .column - flex column container layout. Distributes it's child elements in a column layout.
    3. .reverse - reverse order in layout.
    4. .wrap - wrap child elements.
    5. .nowrap - do not wrap child elements.
    6. .wrap-reverse - wrap child elements in reverse.

    7. .align-start .align-center .align-end .align-stretch - alignment of child elements according to container.

      with .row - changes vertical alignment

      with .column - changes horizontal alignment

    8. .items-start .items-center .items-end .items-stretch .items-space-between .items-space-around .items-space-even

      alignment of child elements according to container.

      with .row - changes horizontal alignment

      with .column - changes vertical alignment

    9. .content-start .content-center .content-end .content-stretch .content-space-between .content-space-around .content-space-even

      alignment of content when child elements are wrapped.

      with .row .wrap - changes vertical alignment

      with .column .wrap - changes horizontal alignment

  • Item Classes

    1. .item - child element of a parent flex container element (.row / .column).

      Gets distributed as per container's layout (row - horizontal / column - vertical)

    2. .grow - expands to take available place in parent container.

      inside .row - expands horizontally

      inside .column - expands vertically

    3. .shrink - shrinks to allow other sibling elements to grow.

      inside .row - shrinks horizontally

      inside .column - shrinks vertically

    4. .nogrow - does not allow element to grow.

    5. .noshrink - does not allow element to shrink.

    6. .start .center .end .stretch - align element according to parent flex container.

      inside .row - changes vertical alignment

      inside .column - changes horizontal alignment

  • Grid System Classes - Always used inside .row .wrap and with .item class.
